# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview
This project is an interactive Hospital Emergency Room Dashboard built in Microsoft Excel to analyze patient flow, operational efficiency, and emergency room performance metrics.

The dashboard provides meaningful healthcare insights through dynamic visualizations, KPI cards, slicers, and analytical charts.

---

# 🎯 Business Problem
Hospitals require real-time operational insights to improve patient experience, reduce emergency room delays, and optimize healthcare resource allocation.

This dashboard helps monitor:
- Patient traffic
- Wait times
- Admission rates
- Referral departments
- Patient satisfaction
- Emergency room efficiency

---

# 📊 Dashboard Features

## KPI Metrics
- Total Number of Patients
- Average Wait Time
- Patient Satisfaction Score
- Admission Rate
- Referral Rate
- Patients Seen Within 30 Minutes

## Analytical Visualizations
- Patient Age Distribution
- Gender Analysis
- Department Referral Analysis
- Admission vs Non-Admission Analysis
- Monthly Performance Insights

# 📈 Key Insights
- Majority of patients belong to the 70–79 age group
- 59% of patients were attended within time
- General Practice received the highest referrals
- Average patient wait time was 35.2 minutes
- Admission and non-admission rates were nearly balanced
